## Recording Tips

1. **Screen Recording**: Use OBS Studio or similar
2. **Resolution**: 1920x1080
3. **Audio**: Clear voice, no background noise
4. **Length**: Keep under 3 minutes
5. **Format**: MP4, H.264 codec

## Checklist Before Recording

- [ ] Backend server running
- [ ] React app running
- [ ] Desktop app ready
- [ ] Sample CSV file ready
- [ ] Test login credentials
- [ ] Clean browser cache
- [ ] Close unnecessary apps
- [ ] Test audio
- [ ] Prepare talking points
